**CSS Foundations**
This folder contains 7 sub folders that'll help cement and solidify CSS understanding and creating visually appealing webpages using CSS.


----
#### **/The Foundations**

* **Concepts:**

  * CSS syntax (selectors, properties, values)
  * Inline, internal, and external CSS
  * The **box model** (margin, border, padding, content)
  * Units: `px`, `%`, `em`, `rem`, `vh`, `vw`
* **Mini builds:**
  * Practice adjusting box sizes and spacing visually.
---

#### **/Selectors & Specificity**

* **Concepts:**

  * Element, class, ID selectors
  * Attribute selectors `[type="text"]`
  * Pseudo-classes (`:hover`, `:focus`, `:nth-child`)
  * Specificity and the cascade
* **Mini builds:**

  * Create a **login form** with hover/focus states on buttons and inputs.
---

* **Concepts:**
  * `display`: block, inline, inline-block, none
  * `position`: static, relative, absolute, fixed, sticky
  * Floats and clearing
* **Mini builds:**

  * Recreate a **basic navbar and hero section** with positioned elements.

---

#### **/Flexbox**

* **Concepts:**

  * `display: flex` and `flex-direction`, `justify-content`, `align-items`
  * `gap`, `flex-wrap`, `flex-grow/shrink/basis`
* **Mini builds:**
  * Build a **responsive card layout** and **pricing section**.
  * Make it adjust automatically when you resize the window.

---

#### **/Grid**

* **Concepts:**

  * `display: grid`, `grid-template-columns`, `grid-template-rows`
  * `gap`, `grid-area`, `repeat()`, `fr` units
  * Combining Grid and Flexbox effectively
* **Mini builds:**
  * A **photo gallery** using grid.
---

#### **/Animations , Transitions and Media Queries**

* **Concepts:**

  * Colors (`rgba`, `hsl`), gradients
  * Shadows, transitions, transforms, animations
  * Custom properties (CSS variables)
  * Responsive design basics with `@media`
* **Mini builds:**

  * Animate a **button hover effect** or **card flip**.

---

#### **/Typography and Review**

* **Final project idea:**
  * A **simple blog layout** (home page + article page)
